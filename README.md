# Homebrew Scotty
This repository contains the homebrew formula for [scotty](https://github.com/wdullaer/scotty).

```sh
brew tap wdullaer/scotty
brew install scotty
```

Formulas are included for the following targets:
* OSX
* 64bit glibc Linux
